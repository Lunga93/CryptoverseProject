Cryptoverse

Cryptoverse is a cryptocurrency tracker that was created by devworks during 2021.The system was designed to give people that are interested in trading cryptocurrecy a user friendly and learnable platform to trade and invest.
We hope that you have a great user experience enagaging with the platform and if there are any uncertainities,please do feel free to contact us .

Functionality

Cryptoverse has been designed to be able to record and display the balances of individual cryptocurrecies that are held.The systems displays these current values in dollars.Moreover the system has been designed to keep a record on all trades made and enables the user to examine their previous trades through descriptive reports.
Above all else the system is heavily centred on security to protect the interests of the user and their information.The security measures put forward require user authentication through the use of passwords and specified usernames.
